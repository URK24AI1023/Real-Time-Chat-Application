# Real-Time Chat Application 

A real-time chat application built using the MERN Stack and Socket.IO that allows multiple users to communicate instantly and stores messages in MongoDB.

##  Features

- User Registration and Login
- Real-time Messaging using Socket.IO
- Multiple User Support
- Chat History Storage in MongoDB
- Responsive Chat Interface
- User Online Status

##  Technologies Used

### Frontend
- React.js
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- Socket.IO

### Database
- MongoDB
- Mongoose

##  Installation

### Clone the Repository

```bash
git clone https://github.com/URK24AI1023/Real-Time-Chat-Application.git
```

### Backend Setup

```bash
cd server
npm install
node server.js
```

### Frontend Setup

```bash
cd client
npm install
npm start
```
